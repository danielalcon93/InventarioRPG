## PROYECTO INVENTARIO RPG

Este proyecto es un sistema de inventario para un juego de ROL en Java.
El sistema permite a los jugadores gestionar sus objetos,
para agregar, eliminar y visualizar los objetos en el inventario.

## Requisitos
- Java 25
- Maven

## Instalación
- Clonar el repositorio
- Ejecutar el comando mvn clean install para compilar el proyecto
- Ejecutar el comando mvn exec:java para iniciar la aplicación

- o Ejecutar el archivo.jar generado en la carpeta resources.
